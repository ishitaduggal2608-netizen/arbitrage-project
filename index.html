About IPL Auction Portal

The IPL Auction Portal is a web-based application designed to simulate a real-time IPL Mega Auction. It allows franchises to log in, view available players, place bids, and build their squads dynamically.


Key Features

  • Franchise login system
  • Live player bidding
  • Dynamic purse deduction
  • Real-time auction updates
  • Team squad management
  • Franchise dashboard overview

Technologies Used

This website is built using HTML for structure, CSS for styling, and JavaScript for dynamic functionality including bidding logic and UI updates.
